## 系统概述

CDK-Office 是一个现代化的企业级办公系统，提供数据隔离、智能审批、知识管理、PDF处理等核心功能。

### 主要特性

- 🔒 **数据隔离**: 严格的团队数据隔离，确保数据安全
- 📋 **智能审批**: 可视化工作流设计，自动化审批流程
- 📚 **知识管理**: 个人和团队知识库，智能内容搜索
- 📄 **PDF工具**: 强大的PDF处理功能，支持20+种操作
- ⚡ **性能优化**: 智能缓存、限流、熔断等优化机制

## 快速开始

### 1. 系统登录

1. 访问系统地址：`https://your-domain.com`
2. 输入用户名和密码
3. 选择登录方式：

### 2. 界面介绍

主界面包含：
- **顶部导航栏**: 主要功能入口和用户信息
- **侧边栏**: 详细的功能菜单
- **主内容区**: 当前功能页面内容
- **状态栏**: 系统状态和快捷操作

### 3. 首次使用设置

1. 完善个人信息（姓名、邮箱、手机号）
2. 加入或创建团队
3. 配置通知偏好

## 数据隔离

### 隔离级别

1. **无隔离**: 所有数据对所有用户可见
2. **基础隔离**: 按部门简单隔离
3. **标准隔离**: 按团队隔离，支持跨团队共享
4. **严格隔离**: 需要明确授权才能跨团队访问
5. **完全隔离**: 团队间完全不可见

### 配置方法

1. 进入"设置" → "数据隔离"
2. 选择隔离级别
3. 配置共享规则和访问控制
4. 设置数据脱敏规则

## 审批流程

### 创建工作流

1. 进入"审批" → "流程设计"
2. 点击"新建流程"
3. 使用可视化设计器创建流程
4. 配置节点类型：开始、用户任务、服务任务、决策节点等
5. 设计表单和配置审批人

### 发起审批

1. 进入"审批" → "发起申请"
2. 选择审批类型
3. 填写申请表单
4. 添加附件（如果需要）
5. 提交申请

### 处理审批

1. 进入"审批" → "待办任务"
2. 查看待处理的审批
3. 点击任务查看详情
4. 选择"同意"或"拒绝"
5. 填写审批意见并提交

## 知识库管理

### 个人知识库

#### 创建知识库

1. 进入"知识库" → "个人知识库"
2. 点击"新建知识库"
3. 设置知识库信息：
   - 名称：给知识库起名
   - 描述：简要说明用途
   - 类型：文档、笔记、聊天记录等
   - 隐私设置：公开、团队内或私有

#### 上传文档

支持多种方式：
- **拖拽上传**: 直接将文件拖拽到页面
- **批量上传**: 选择多个文件同时上传
- **微信聊天记录**: 上传txt格式的聊天记录

### 智能搜索

1. **关键词搜索**: 输入关键词快速查找
2. **高级搜索**: 按类型、时间、标签等筛选
3. **AI智能问答**: 基于知识库内容的问答

### 团队协作

1. **分享文档**: 将个人文档分享到团队
2. **协作编辑**: 多人同时编辑文档
3. **评论讨论**: 在文档中添加评论和讨论

## PDF工具

### 支持的操作

系统集成Stirling PDF，支持20+种操作：

#### 合并和拆分
- 合并PDF：将多个PDF合并为一个
- 拆分PDF：按页数或书签拆分
- 提取页面：提取指定页面

#### 转换功能
- PDF转图片：转为JPG、PNG等格式
- 图片转PDF：将图片转换为PDF
- PDF转Word：转换为可编辑文档

#### 安全操作
- 添加密码：为PDF设置密码保护
- 数字签名：添加数字签名
- 权限设置：设置打印、复制权限

#### 优化功能
- 压缩PDF：减小文件大小
- 添加水印：添加文字或图片水印
- OCR识别：识别PDF中的文字

### 使用方法

1. 进入"PDF工具"
2. 上传PDF文件
3. 选择要执行的操作
4. 设置操作参数
5. 开始处理并下载结果

## 常见问题

### 登录问题

**Q: 忘记密码怎么办？**
A: 在登录页面点击"忘记密码"，按提示重置密码。

**Q: 无法登录系统？**
A: 检查用户名密码、账户状态、网络连接和浏览器兼容性。

### 权限问题

**Q: 提示没有访问权限？**
A: 可能是角色权限不足、资源私有或数据隔离限制，联系管理员获取权限。

**Q: 无法查看其他团队文档？**
A: 这是正常的数据隔离机制，需要申请跨团队访问权限。

### 文件处理问题

**Q: 文件上传失败？**
A: 检查文件大小（默认50MB限制）、格式支持、网络连接和存储空间。

**Q: PDF处理失败？**
A: 可能是文件损坏、格式不支持或处理超时，尝试重新上传或联系支持。

### 审批流程问题

**Q: 审批流程卡住了？**
A: 联系当前审批人或使用委派功能，管理员可强制流转。

**Q: 如何撤销申请？**
A: 在"我的申请"中找到申请，如果状态允许可点击"撤销"。
