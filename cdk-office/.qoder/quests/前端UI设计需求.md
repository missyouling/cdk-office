---

### 🎨 总体设计原则
- **技术栈**：Next.js + React 19 + TypeScript + Tailwind CSS + Shadcn UI
- **风格**：现代简约、企业级专业感、高可用性、响应式设计
- **交互**：流畅动效、清晰反馈、无障碍支持
- **主题**：支持深色/浅色主题，色彩系统与CDK-Office保持一致

---

### 1. 智能问答系统 UI 提示词

**目标**：打造一个直观、响应迅速的AI对话界面，支持多轮问答、来源追溯、反馈机制。

**提示词**：
```
设计一个企业级AI问答界面，包含：
- 左侧边栏：历史会话列表，支持按时间/主题筛选
- 主区域：对话气泡布局，用户问题居右，AI回答居左，支持Markdown渲染
- 回答底部显示来源文档链接（可点击跳转）和置信度标签
- 底部固定输入栏：支持多行输入，附发送按钮和附件上传功能
- 反馈机制：每个回答下方提供“👍/👎”反馈按钮，点击后可输入详细反馈
- 响应式设计：在移动端折叠边栏，主区域全屏显示
- 使用Shadcn UI的Card、Input、Button、Avatar组件
- 主题色：蓝色系，符合企业品牌色
```

**提示词**：
