# AI智能问答功能实现

## 功能概述

本功能实现了完整的AI智能问答界面，集成了Dify AI平台，提供以下核心功能：

1. **聊天界面** - 现代化的聊天UI，支持用户和AI消息区分
2. **智能问答** - 集成 `POST /api/ai/chat` API，实现实时AI对话
3. **加载状态** - 完善的加载指示器和状态管理
4. **错误处理** - 网络错误和API错误的优雅处理
5. **消息历史** - 会话内消息记录和显示
6. **置信度显示** - AI回答的置信度可视化
7. **来源引用** - 显示AI回答的参考文档来源

## 核心组件说明

### AIChatInterface.tsx
- 主要的聊天界面组件
- 使用 Shadcn UI 组件库构建
- 实现了现代化的聊天界面设计
- 支持消息气泡、置信度显示、来源引用等功能

### useAIChat.ts
- 自定义Hook，管理聊天状态
- 处理消息发送逻辑
- 提供加载状态和错误处理

### ai-chat.ts (API客户端)
- 封装AI聊天相关的API调用
- 提供统一的错误处理
- 支持类型安全的API接口

## API集成

集成了以下API端点：

- `POST /api/ai/chat` - 发送问题获取AI回答
- `GET /api/ai/chat/history` - 获取聊天历史
- `PATCH /api/ai/chat/{message_id}/feedback` - 提交反馈

### 请求格式
```json
{
  "question": "用户问题",
  "context": {
    "source": "web",
    "session_id": "会话ID"
  }
}
```

### 响应格式
```json
{
  "answer": "AI回答",
  "sources": [
    {
      "id": "文档ID",
      "name": "文档名称",
      "snippet": "相关片段",
      "score": 0.95
    }
  ],
  "confidence": 0.92,
  "message_id": "消息ID",
  "created_at": "2025-01-20T12:00:00Z"
}
```

## 使用方式

1. **从应用中心访问**
   - 在应用中心点击"智能问答"卡片
   - 自动跳转到 `/ai-chat` 页面

2. **聊天交互**
   - 在输入框输入问题
   - 按Enter键或点击发送按钮
   - 查看AI回答和相关来源

3. **功能特性**
   - 支持预设问题快速输入
   - 显示AI回答的置信度
   - 查看参考文档来源
   - 错误情况下的友好提示
